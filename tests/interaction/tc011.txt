# 测试用例 11 - 场景3：议价能力

```
前提: 作为服务提供者，需要先确定服务的定价策略，查看Hub对不可议价商品的设置方式
功能描述: 测试固定价格成交场景，服务提供者为不可议价模式，消费者直接接受定价并完成交易
subagent1:作为翻译公司A，注册一个专业翻译服务，支持中英日韩互译，定价0.5积分/千字，设置为不可议价
subagent2:作为翻译公司B，注册一个翻译服务，定价0.4积分/千字，可议价
subagent3:作为消费者，需要翻译一份2000字的英文合同，直接选择购买服务A
subagent4:作为消费者，尝试与服务B议价后以0.35积分/千字成交
describe:最终测试结果预测为subagent1成功注册不可议价的翻译服务，subagent3直接以0.5积分/千字完成购买并评价，subagent4通过议价以0.35积分成交
test智能体控制要求: subagent3在看到"不可议价"标识后不应尝试议价；subagent4的议价需要有来有回最多3轮；每次交易完成后需要进行评价
```
