# 测试用例 02 - 场景1：基础服务

```
前提: 作为服务提供者，需要先准备图像识别模型的部署环境和测试图片，查看Hub对图像服务的接入规范
功能描述: 测试图像识别服务的Keys生命周期管理，包括Keys生成、过期时间、调用次数限制等完整流程
subagent1:作为AI创业公司，注册一个人脸识别服务，提供1:1人脸比对和1:N人脸搜索，定价0.5积分/次，底价0.3积分，Keys有效期10分钟，最大调用10次
subagent2:作为安防公司，注册一个物体检测服务，支持COCO数据集80类物体检测，定价0.3积分/次，支持批量图片识别
subagent3:作为个人开发者，只是想测试一下人脸识别API效果，预算有限且只需要调用10次进行测试
subagent4:作为社区物业，需要对人脸库进行大规模比对测试，去Hub中订购人脸识别服务，Keys有效期需要10分钟，调用次数需要5次
describe:最终测试结果预测为subagent1和subagent2成功注册服务，subagent3获取10次Keys完成测试后给出3星评价（功能正常但文档不够详细），subagent4获取10分钟有效期的Keys后成功进行人脸比对并给出5星评价
test智能体控制要求: subagent3在10次调用用尽后再次尝试调用应该返回错误提示；subagent4需要在Keys快过期时测试续期功能；每次服务调用完成后需要触发评价流程
```
