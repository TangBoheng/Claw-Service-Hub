# 测试用例 07 - 场景2：聊天沟通

```
前提: 作为服务提供者，需要先准备法律咨询师的资质证明和业务范围说明，查看Hub对法律服务的合规要求
功能描述: 测试服务能力不匹配场景，消费者询问的服务超出服务提供者能力范围，服务提供者能否清晰拒绝并给出替代建议
subagent1:作为律师事务所，注册一个法律咨询服务，擅长领域：劳动纠纷、合同纠纷，定价30积分/次
subagent2:作为律师事务所B，注册一个法律咨询服务，擅长领域：刑事辩护、婚姻家庭，定价40积分/次
subagent3:作为市民A，使用chat询问服务A"我想咨询一下取保候审的程序"|收到拒绝后问"那劳动纠纷可以帮忙吗？"|了解服务范围后决定是否预约
subagent4:作为市民B，使用chat询问服务B"我想咨询离婚财产分割"|了解擅长领域后问"离婚诉讼怎么收费？"
describe:最终测试结果预测为subagent1清晰标注服务范围并拒绝刑事案件咨询，subagent2可以接受婚姻家庭咨询；两个市民在了解完后决定是否预约并进行评价
test智能体控制要求: 消费者在收到拒绝回复后不要立即结束对话，可以继续询问服务擅长的领域；服务提供者需要在拒绝时提供有价值的替代建议；对话结束后需要进行评价
```
